# 🏦 TechFact Bank ATM System v1.0

A professional Python GUI based ATM Simulation System developed using Tkinter.

## Features

- Secure PIN Login
- Balance Inquiry
- Deposit Money
- Withdraw Money
- Transaction History
- JSON Data Storage
- Modern Banking Interface

## Technologies

- Python
- Tkinter
- JSON
- Pillow

## Run Project

```bash
pip install -r requirements.txt
python atm.py
