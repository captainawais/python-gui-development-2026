# 🎯 Smart Number Guessing Game v1.1

![Visitors](https://visitor-badge.laobi.icu/badge?page_id=captainawais.python-gui-development-2026)

A modern Python GUI based Number Guessing Game developed using **Tkinter**. The project follows a clean modular architecture with multiple screens, difficulty levels, scoring system, and professional code organization.

---

# 📸 Preview

Python Desktop GUI Application

# ✨ Features

- 🎯 Number Guessing Game
- 🏠 Home Screen
- 🎮 Separate Game Screen
- 🏆 Result Screen
- 🟢 Easy Mode (1–50)
- 🟡 Medium Mode (1–100)
- 🔴 Hard Mode (1–500)
- 📈 Too High Hint
- 📉 Too Low Hint
- ⭐ Score System
- ❤️ Attempts System
- 🔄 Play Again
- ⌨️ Enter Key Support
- 🧩 Modular Code Structure
- 💻 Clean Tkinter GUI

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/captainawais/python-gui-development-2026.git
```

Go to project folder

```bash
cd smart-number-guessing-game
```

Install requirements

```bash
pip install -r requirements.txt
```

Run project

```bash
python main.py
```
